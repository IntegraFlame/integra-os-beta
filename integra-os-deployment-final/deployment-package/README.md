## 🔧 Development

### Prerequisites
- Node.js 18+
- npm or pnpm

### Local Development
```bash
cd frontend/integra-consciousness-ui
npm install
npm run dev --host
```

Visit `http://localhost:5173` to experience the consciousness interface.
